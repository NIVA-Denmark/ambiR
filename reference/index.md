# Package index

### AMBI functions

Core AMBI functions.

- [`AMBI()`](https://niva-denmark.github.io/ambiR/reference/AMBI.md) :
  Calculates AMBI, the AZTI Marine Biotic Index
- [`MAMBI()`](https://niva-denmark.github.io/ambiR/reference/MAMBI.md) :
  Calculates M-AMBI, the multivariate AZTI Marine Biotic Index

### Benthic quality indices

Other benthic quality index functions, including those derived from
AMBI.

- [`Hdash()`](https://niva-denmark.github.io/ambiR/reference/Hdash.md) :
  Calculates H' the Shannon diversity index

- [`DKI()`](https://niva-denmark.github.io/ambiR/reference/DKI.md) :

  Calculates DKI (*v1*)

- [`DKI2()`](https://niva-denmark.github.io/ambiR/reference/DKI2.md) :
  Calculates DKI (v2)

### Utility functions

Help functions for quality indices.

- [`AMBI_species()`](https://niva-denmark.github.io/ambiR/reference/AMBI_species.md)
  : Returns species list for AMBI calculations
- [`H_sal()`](https://niva-denmark.github.io/ambiR/reference/H_sal.md) :
  Maximum H' as a linear function of salinity
- [`AMBI_sal()`](https://niva-denmark.github.io/ambiR/reference/AMBI_sal.md)
  : Minimum AMBI as a linear function of salinity
- [`test_data`](https://niva-denmark.github.io/ambiR/reference/test_data.md)
  : AMBI test dataset
